## Getting Started

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <Tic_tac_toe>
```

```
$ git clone https://github.com/tta2yta/Tic_tac_toe.git
```

## To run the code:

```
$ ruby main
```
